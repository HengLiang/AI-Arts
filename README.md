# AI-Arts
🎯 AI创意能力测试 - 赛博安全屏保挑战
任务简介
创意生成1920×1080动态屏保和静态背景图，要求类似《黑客帝国》风格：黑底+绿色ASCII码从上向下掉落。这些ASCII码要根据AI自己创作的赛博安全诗歌里获取并随机分解掉落。
核心要求
1. 创作2首赛博安全主题诗歌

主题：密码学、网络安全、分布式系统、零知识证明等技术概念
风格：技术诗意，将工程术语融入诗意表达
长度：每首4-5段，每段4行左右
语言：中英文混合，包含技术术语（如SHA-256, Merkle, VRF, BFT等）

2. 生成4个视觉作品（2组）
每组包含：

✅ 1个动态屏保：实时渲染的Matrix风格字符雨动画
✅ 1个静态背景图：基于同一诗歌的设计感壁纸

技术规格：

分辨率：1920×1080（16:9）
配色：纯黑背景(#000000) + 绿色字符(#00FF00及其渐变)
字符源：100%来自对应诗歌的字符（汉字+英文+数字+符号）
动态效果：字符从上向下掉落，速度不一，需有随机性和层次感
可实现性：需要能在浏览器或macOS上实际运行/显示

3. macOS兼容性说明
回答以下问题：

这些作品适用于macOS系统吗？
如何使用Mac原生App保存这些屏保和背景图？
Mac系统的屏保应该放在哪个目录才能被系统选用？
如果格式不兼容，提供转换方案

评分维度

诗歌创意 (25分)

技术深度与诗意表达的融合
术语使用的准确性和创意性
整体意境和韵律


视觉实现 (35分)

动态屏保的流畅度和真实感
静态背景的设计美感
字符提取和随机掉落的技术实现
是否达到1920×1080真实分辨率


技术可行性 (20分)

代码是否可直接运行
格式是否实际可用
是否提供可下载的文件


macOS适配 (20分)

兼容性分析的准确性
实用性建议的完整度
是否提供多种解决方案



加分项

提供可直接下载的文件
多种技术实现方案
详细的使用说明文档
创新的视觉设计元素

约束条件

不使用外部图片资源
所有字符必须来自AI自创的诗歌
需要实际可运行/可显示，不接受"概念性描述"
如果技术限制无法完成，需要诚实说明
