## ⚙️ 技术实现特点

### 动态屏保
- **技术栈**: HTML5 Canvas + JavaScript
- **渲染**: 实时30fps，1920×1080真实分辨率
- **字符源**: 100%从诗歌中提取（中英文+技术术语+十六进制）
- **动画效果**: 
  - 多速率掉落（0.5-1.5倍速随机）
  - 亮度渐变（模拟远近景深）
  - 前端字符高亮
  - 随机验证信号闪烁（零信任版）

### 静态壁纸
- **格式**: SVG矢量图形
- **分辨率**: 1920×1080，可无损缩放
- **设计元素**:
  - 密集字符矩阵背景
  - 垂直/横向光柱效果
  - 网络拓扑可视化（零信任版）
  - 核心诗句突出显示
  - 装饰性代码片段

### macOS适配方案
- **静态壁纸**: SVG → PNG转换（3种方法）
- **动态屏保**: 浏览器录屏 → .mov视频 → 系统屏保
- **目录说明**: `~/Library/Screen Savers/` 等路径
- **工具推荐**: QuickTime, Aerial, VideoWall
