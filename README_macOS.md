# 赛博安全壁纸 - macOS使用指南

## 📦 文件清单

本包包含2组Matrix风格赛博安全壁纸：

1. **数据熵墙** (entropy_wall_1920x1080.svg)
2. **零信任协定** (zero_trust_1920x1080.svg)

## 🖼️ 静态壁纸使用方法

### 方法1：直接设置（推荐）

1. **打开SVG文件**
   - 双击SVG文件，在Safari或Preview中打开
   
2. **截图保存为PNG**
   - macOS Sonoma/Ventura: `Shift + Command + 4`，然后按`空格键`
   - 将光标移到窗口上，点击截图
   - 图片会自动保存到桌面
   
3. **设置为壁纸**
   ```
   系统设置 → 墙纸 → 点击"+"号 → 选择截图的PNG
   ```

### 方法2：转换为PNG（高质量）

使用macOS自带的预览(Preview)转换：

1. 用Preview打开SVG文件
2. `文件 → 导出...`
3. 格式选择"PNG"
4. 分辨率选择"实际大小"
5. 保存后设置为壁纸

### 方法3：使用浏览器

1. 用Safari打开SVG文件
2. 右键点击 → "将图像存储为..."
3. 保存为PNG格式
4. 设置为壁纸

## 🎬 动态屏保使用方法

### 录制网页动画为视频

macOS无法直接使用HTML动画作为屏保，需要转换为视频：

1. **在浏览器中打开动态屏保网页**
   
2. **使用QuickTime录屏**
   ```
   应用程序 → QuickTime Player → 文件 → 新建屏幕录制
   ```
   - 录制30-60秒的循环动画
   - 停止录制后保存为.mov文件

3. **设置视频为屏保**
   ```
   系统设置 → 屏幕保护程序 → 选择器 → 点击"+"
   ```
   - 然后选择录制的.mov视频文件

### 使用截图工具（macOS Mojave及更高版本）

```
Shift + Command + 5
```
- 选择"录制所选部分"
- 框选浏览器窗口区域
- 点击"录制"
- 录制30-60秒后停止
- 视频自动保存到桌面

## 📁 macOS路径参考

### 壁纸
- 直接通过"系统设置 → 墙纸"选择即可
- 或放到：`~/Pictures/` 文件夹

### 屏保
- 用户级：`~/Library/Screen Savers/`
- 系统级：`/Library/Screen Savers/`（需管理员权限）

**注意**：macOS原生屏保需要`.saver`格式，但可以通过"屏幕保护程序"设置选择视频文件。

## 🔧 高级选项

### 使用第三方工具

- **Aerial**：支持视频屏保，可以更好地循环播放
  - 下载：https://aerialscreensaver.github.io/
  
- **VideoWall**：将视频设为动态壁纸
  - Mac App Store搜索下载

### 转换SVG为高分辨率PNG（命令行）

如果你熟悉Terminal：

```bash
# 使用Homebrew安装ImageMagick
brew install imagemagick

# 转换SVG为PNG
convert -density 300 entropy_wall_1920x1080.svg entropy_wall.png
```

## 🎨 显示效果优化

- **分辨率**：1920×1080 适合大多数显示器
- **颜色配置**：建议显示器色彩模式设为"标准"或"sRGB"
- **亮度**：赛博风格建议在暗环境下观看效果更佳
- **对比度**：可在"系统设置 → 显示器"中微调

## ⚠️ 注意事项

1. SVG文件在不同浏览器/应用中可能显示效果略有差异
2. 动态效果需要录制成视频才能作为macOS屏保
3. OLED屏幕长时间显示纯黑背景可节省电量
4. 动态屏保可能消耗更多系统资源

## 📞 技术支持

如有问题，可以：
- 检查SVG文件是否完整下载
- 尝试不同的转换方法
- 确保macOS版本在10.14 (Mojave)或更高
